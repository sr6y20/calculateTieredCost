# 分段费用计算器

一个简单易用的分段费用计算器，支持三档费率的自动计算。

## 功能特点

- 📊 支持三档分段计费
- 🎯 实时计算结果
- 💻 响应式设计
- ⚡ 输入验证
- 📱 移动端适配

## 费用档位说明

| 档位   | 金额范围   | 折扣率               |
| ------ | ---------- | -------------------- |
| 第一档 | 0-80 元    | 原价（100%）         |
| 第二档 | 80-200 元  | 超出 80 元部分 8 折  |
| 第三档 | 200 元以上 | 超出 200 元部分 5 折 |

## 使用方法

1. 输入单次费用金额
2. 输入天数（每天计算两次）
3. 点击"计算费用"按钮或按回车键
4. 查看计算结果

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)

## 快速开始

1. 克隆或下载项目文件
2. 在浏览器中打开 `index.html`
3. 开始使用计算器

## 示例

```javascript
// 计算示例
const cost = 3; // 单次费用3元
const days = 22; // 22天
const total = calculateTieredCost(cost, days); // 结果：88.0元
```
